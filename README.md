# 一个基于kotlin+MVVM+jetpack+ViewBinding打造的玩Android项目

## 项目结构

- kotlin语言
- MVVM结构
- jetpack组件
- ViewBinding
- Retrofit + 协程
- Glide 图片加载
- swiperefreshlayout 实现下拉刷新

## 实现功能

- 首页
- 公众号
- 体系、导航
- 我的、登录注册
